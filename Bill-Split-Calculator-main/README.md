## 🧠 Core Formula

Each Person Pays = Total Bill ÷ Number of People

share = (amount / people).toFixed(2)

✔ Divides total equally  
✔ Rounds to 2 decimal places  
✔ Keeps currency format clean  

---

## ⚙️ How It Works

1️⃣ User enters Total Bill  
2️⃣ User enters Number of People  
3️⃣ Values converted using parseFloat()  
4️⃣ Validation using !isNaN() and > 0  
5️⃣ Formula applied → amount / people  
6️⃣ Result displayed using innerText
