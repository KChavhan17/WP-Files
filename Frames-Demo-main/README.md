## 🖼️ Frames Demo (HTML)


## 📌 Overview


This one  demonstrates the use of the
HTML Frameset layout.


Frames divide a webpage into
multiple sections and load different 
HTML files in each section.



## 📂 Files:


- "index.html" – Main frameset layout

  
- "top.html" – Top frame content

  
- "left.html" – Left frame content

  
- "right.html" – Right frame content

  
---


## ⚙️ How It Works


The "index.html" file creates the layout using framesets.


- The page is divided into two rows (30% and 70%)

  
- The top row loads "top.html"

  
- The bottom row is divided into two columns (50% and 50%)

  
- The left column loads "left.html"

  
- The right column loads "right.html"

  
🧩 This creates a simple 3-frame webpage layout.


---


## ⚠️ Important Note


HTML Frames and Framesets are deprecated in HTML5 
and are no longer recommended for modern websites.


They are included here only for learning purposes.
