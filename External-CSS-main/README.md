## 🎨 External CSS: The Clean Way 🚀


It shows how to keep HTML and CSS separate. 


This makes the code professional and very easy to read.


## 📂 How it works


### 1. index.html (The Structure)


🧰This file is the "Skeleton." It only contains the text and the buttons.


⛓️The Bridge: I used the <link> tag to connect the CSS.


🛠️Tags used: `<div>` for the box, 


`<h1>` for the title, 


and  `<button>`  for the action.


### 2. style.css (The Design)


🎨This file is the "Paint." It handles all the colors and the looks.


🎨Flexbox: Used to put everything perfectly in the middle. 📏


🎨Dark Mode: Used a black background for a cool look. 🌙


🎨Neon Glow: Used box-shadow to make the box shine.


🎨Hover: The button grows when you touch it. 🔘


## 🛠 Languages Used


HTML5: For content and structure. 🧱


CSS3: For styling and animations. ✨
