## ⚙️ Attributes Used


border="2" – Table border


cellpadding="8" – Space inside each cell


cellspacing="0" – No gap between cells


align="center" – Applied in `<tr>` to center all subjects horizontally


colspan="6" – Used for LEISURE row


rowspan="3" – Used for lab sessions


valign="middle" – Applied only to the three lab cells


## 🧠 Alignment Understanding
align is given in `<tr>` (table row).


This automatically centers all subjects in that row.


valign="middle" is used only for:


CN LAB


DAA LAB


WP LAB


Because these use rowspan="3" and need vertical centering.


## 🎯 Learning Outcome


Row-level alignment control


Horizontal merging (colspan)


Vertical merging (rowspan)


Vertical alignment handling for tall cells
